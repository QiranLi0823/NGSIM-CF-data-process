# 特征变量计算分析报告

## 1. 概述

本报告描述了跟驰场景下特征变量的计算过程和统计结果。

## 2. 特征定义

| 特征 | 公式 | 说明 |
|------|------|------|
| 后车速度 (v_f) | 直接提取 | 平滑后的后车速度 (m/s) |
| 相对距离 (s) | s = y_leader - y_follower - L_leader | 前后车车身间距 (m) |
| 相对速度 (v_rel) | v_rel = v_leader - v_follower | 前车速度 - 后车速度 (m/s) |
| 标签 | 后车加速度 | 用于模型预测的目标变量 (m/s²) |
