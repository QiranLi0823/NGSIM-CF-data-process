# 模型验证分析报告

## 1. 概述

本报告描述了使用LSTM模型对处理后的NGSIM-US-101数据集进行验证的结果。

## 2. 模型架构

### 2.1 网络结构

| 组件 | 配置 |
|------|------|
| 模型类型 | LSTM |
| 输入特征 | 3 (spacing, v_rel, v_follower) |
| 隐藏层大小 | 64 |
| LSTM层数 | 2 |
| 输出 | 1 (加速度) |

### 2.2 训练参数

| 参数 | 值 |
|------|------|
| 批次大小 | 32 |
| 学习率 | 0.001 |
| 训练轮数 | 20 |
| 优化器 | Adam |

## 3. 评估结果

| 指标 | 数值 |
|------|------|
| MSE (均方误差) | 0.0184 |
| MAE (平均绝对误差) | 0.0732 |
| R2 (决定系数) | 0.9766 |

## 4. 训练过程

训练损失从 0.6129 变化到 0.0221
测试损失从 0.4214 变化到 0.0184

## 5. 结论

1. 模型成功完成训练，表明数据集可用于深度学习模型训练
2. R2为 0.9766，表明模型具有一定预测能力
3. 数据处理流程完整，格式正确
